# 2026-07-15 — Milestone 5.2: env.GetDefault — Milestone 5 complete

## What got done

- **Roadmap 5.2 `env.GetDefault`** (branch `feat/env-getdefault`): the typed-env-reads utility,
  the second and final item of Milestone 5 — **M5 (Configuration & environment) is complete**.
  Package `env` with `GetDefault(key, fallback string) string` plus `GetInt`, `GetBool`, and
  `GetDuration` typed variants (spec §5 line 108).
- **Uniform "safe fallback" contract** (spec §2.14): each getter returns the parsed value only
  when the variable is set to a non-empty, well-formed string; an **unset, empty, or malformed**
  value all yield the caller's fallback. No error is returned — a bad value is treated as "not
  configured". `GetInt` uses `strconv.Atoi`, `GetBool` `strconv.ParseBool`, `GetDuration`
  `time.ParseDuration`.
- **No ADR** — this is routine implementation with no pattern adoption, security-relevant choice,
  or non-obvious one-way decision (ADR policy, `docs/adr/README.md`). The only judgment call
  (empty ⇒ fallback, consistent across all getters) is documented in the godoc.
- Local gauntlet green (portable Go 1.26.5): build, vet, full `go test ./...`, **100% env
  coverage**, gofumpt clean, golangci-lint v2 (`/v2` path) 0 issues, govulncheck clean,
