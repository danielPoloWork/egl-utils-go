# Spec v2.0 — imported draft (provenance)

The files in this folder are a **verbatim import** of the spec-review v2.0 draft of `d4np-go`
(`d4np-go.md`, "Reviewed draft", 2026-07-14, resolving spec-review issue #8) and its three
accompanying ADRs, which previously lived only in the untracked `.spec/` folder at the repo root.
They are imported unmodified so future spec revisions land inside the governed tree — the
recommendation of the [gap analysis](../02_spec_v2_gap_analysis.md) §7.

**Normative status:** this draft does **not** replace the intake-frozen spec
([`../01_spec_utils.md`](../01_spec_utils.md)), which remains the contract v1.0.0 was built and
verified against. The disposition of every v2 delta — adopted in v1.x, deferred to a `/v2` major,
or maintained as a documented deviation — is recorded in
[ADR-0030](../../adr/0030-spec-v2-reconciliation.md) and tracked as **Milestone 10** in the
[ROADMAP](../../../ROADMAP.md).

Do not edit these files; a future spec revision arrives as a new import with its own
reconciliation.
